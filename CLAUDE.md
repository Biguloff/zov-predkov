# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Is

This is a project folder for "Зов Предков | ИИ проект" (Call of the Ancestors) — an AI-generated video project. It is **not a software codebase**. There are no build commands, tests, or development tooling.

## Contents

- **Созвон от пятницы 3 апреля.md** — meeting notes (in Russian) from a creative review session. Contains feedback on video direction: ultra-realistic style, static cinematic shots, specific scene ideas (Уастырджи, molecular zoom, cosmic zoom-out, Балсаг's wheel).
- **Телеграм чат/** — exported Telegram group chat ("Зов Предков | ИИ проект") between Сергей Бигулов, Вова Бирагов, and Заур Соскиев (сценарист). Includes messages, voice messages (.ogg), a scenario PDF, and audio files. The HTML/CSS/JS files are auto-generated by Telegram's export — not custom code.

## Key Context

- The project involves AI-generated video with a focus on Ossetian mythology and culture.
- The creative pipeline: scenario writing -> image generation -> video generation -> sound design.
- Work is done in linear segments (e.g., first 5 seconds completed and approved before moving on).
- Primary language of all materials is Russian.
